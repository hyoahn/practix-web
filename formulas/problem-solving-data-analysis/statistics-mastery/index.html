Standard Deviation Logic

Spread = Gap

Standard Deviation measures how "spread out" data is from the mean. On the SAT, you never calculate it; you only compare it.

The Golden Rule:

Wider range/frequency at edges = Higher SD.
Tight cluster near mean = Lower SD.
