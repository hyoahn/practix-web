Poly-Solve Variables

Solve for k when you know x.

‚ö° Trick: The Regression Solver

The Move: Don't use =. Use ~ (tilde) to make Desmos solve for any letter.

18 ~ 4(3) + 2k

üìù SAT Problem

If \(4x + 2k = 18\) and \(x = 3\), what is the value of \(k\)?

(A) 2
(B) 3
(C) 4.5
(D) 6

Why the original failed?

If you type 4x + 2k = 18, Desmos gets confused because it expects a graph of x and y.

The Regression Fix (~)

By swapping = for ~, you tell Desmos: "Find the value of k that makes this true."

It gives you k = 3 instantly.

‚úÖ Final Answer:

3 (Choice B)
