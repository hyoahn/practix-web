Residual Plots

Visually check if a linear model is appropriate.

üìâ Residual Logic

Random means Good. Pattern means Bad.

Random Scatter = Good Model.

Pattern (U-shape) = Bad Model (Wrong type).

üìù SAT Problem

A data set is fit with a linear model. The residual plot shows a clear U-shaped pattern of points.

What does this pattern indicate about the linear model?

(A) It is a perfect fit
(B) The data is random
(C) A non-linear model is needed
(D) The correlation is zero

Step 1: Check for Patterns

We see a "U-shaped" pattern in the residuals.

Step 2: Interpret the Pattern

Random Noise = Good Linear Fit.

Distinct Pattern = Bad Linear Fit (needs curve).

Step 3: Conclusion

Since there is a pattern, the linear model is not sufficient. We likely need a quadratic or exponential model.
