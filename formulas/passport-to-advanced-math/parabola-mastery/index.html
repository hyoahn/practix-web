Vertex-Line Intersection

\[ k = c - \frac{b^2}{4a} \]

Instantly find the y-value where a horizontal line \(y=k\) is tangent to a parabola \(y=ax^2+bx+c\).

üìù Example: Tangent Line

For \(y = x^2 - 8x + 20\), find the horizontal line tangent to the vertex.

\(k = 20 - \frac{(-8)^2}{4(1)} = 20 - \frac{64}{4} = 4\)

Result: y = 4.

The Vertex Shortcut

\[ x = -\frac{b}{2a} \]

The universal shortcut to find the x-coordinate of the vertex (pivotal point) of any parabola.

üß™ The Proof: Symmetry

The roots of \(ax^2 + bx + c = 0\) are \(\frac{-b \pm \sqrt{b^2-4ac}}{2a}\). The vertex is exactly in the middle of the roots.

\(x_{vertex} = \frac{(\frac{-b + \sqrt{D}}{2a}) + (\frac{-b - \sqrt{D}}{2a})}{2} = \frac{-2b/2a}{2} = -\frac{b}{2a}\)
